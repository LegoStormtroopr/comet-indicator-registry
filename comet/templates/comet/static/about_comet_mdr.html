Comet Health MetaData Registry

The Comet Health MetaData Registry is an extension set for the Aristotle MetaData Registry that provides additional items necessary for tracking the performance of entities in the health context.

Comet adds the following administered items:

  • Data Sources
  • Indicators
  • Indicator Sets
  • Outcome Areas
  • Quality Statements
Comet also adds the following non-administered items:
  • Frameworks
